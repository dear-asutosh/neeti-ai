# Neeti AI 🏛️

An AI Co-Pilot for Public Leaders & Administrators.

## What it does
- Summarizes documents and meeting recordings
- Drafts speeches and official responses
- Manages schedules and events
- Tracks constituency and community data
- Provides real-time AI-powered insights

## Tech Stack
- **Frontend:** React + Vite
- **Database & Auth:** Firebase (Firestore + Authentication)
- **AI:** Groq API (LLaMA for text, Whisper for audio transcription)
- **Charts:** Recharts
- **Styling:** Tailwind CSS
- **Routing:** React Router DOM
